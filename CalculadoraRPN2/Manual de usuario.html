Introduccion

Este programa se ha dise√±ado para su uso como calculadora de expresiones en notacion polaca inversa. Se ha programado en lenguaje Java y puede ejecutarse en cualquier sistema operativo que tenga instalada la Maquina virtual de Java. Para lanzarlo es necesario ejecutar el archivo Calculadora.java.

Menu de opciones

La interfaz del programa ofrece al usuario un menu de opciones por medio del cual podra acceder a las funciones del programa.

A contuniacion de detallas las opciones disponibles en el menu asi como el funcionamiento de cada una de ellas:

  • Ingresar Expresion: Permite al usuario ingresar la expresion de notacion polaca inversa.
  • Imprimir Historial: Permite al usuario acceder al historial de las operaciones realizadas en la calculadora.
  • Salir: Termina y cierra completamente el programa.

Ingresar Expresion (1)

Para ingresar una expresion en la calculadora se debe utilizar notacion polaca inversa; unicamente acepta las 4 operaciones aritmeticas basicas (suma, resta, multiplicacion y division) y los simbolos aritmeticos se deben representar de acuerdo a la siguiente lista de expresiones:

Signo de suma (+) : PLUS
Signo de resta (-) : LESS
Signo de multiplicacion (*) : TIMES
Signo de division (/) : DIVIDE

Imprimir Historial (2)

Esta opcion muestra todas las operaciones realizadas dentro del programa durante la sesion actual, tanto expresiones como resultados.

Salir (3)

Al seleccionar esta opcion el programa se cierra automaticamente.
